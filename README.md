# Jamii App

## Community Emergency Response & Digital Community Platform

**Organization:** Imarisha Jamii

### Vision
To build a smart, technology-driven emergency response platform that connects citizens with emergency responders in real time.

### Features
- Fire Reporting
- Medical Emergency Reporting
- Crime Reporting
- Flood Reporting
- Road Accident Reporting
- Digital Nyumba Kumi
- Community Alerts
- GPS Location
- Photo & Video Upload
- Emergency Notifications

### Technology Stack
- React
- TypeScript
- Vite
- Firebase
- Firestore
- Firebase Authentication
- Firebase Storage
- Cloud Messaging
- Google Maps
- GitHub Codespaces
- Cloudflare
